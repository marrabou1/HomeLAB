# Homelab Stuff

This repository contains scripts, configs, and notes from my personal homelab.

It’s mainly a place for me to keep things I use or experiment with, but feel free to look around or reuse anything that’s helpful.

## What’s in here

You’ll find a mix of things like:

- Small automation scripts
- Docker / container examples
- Service setup helpers
- Config files and templates
- Random homelab experiments

Some things are polished, others are quick hacks.

## Notes

- This is all used in my own setup
- Things may change or break without warning
- Assumptions about OS, paths, or tools are common
- Always read scripts before running them

Use at your own risk.

## Usage

There’s no single way to use this repo.  
Each script or folder should be treated on its own and usually includes comments explaining what it does.
